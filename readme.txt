Controls:
Camera:
W -Up
A -Down
S -Left
D -Right
N -Zoom in
M -Zoom out

Lightball:
J: +ve x-axis
L: -ve x-axis
I: -ve z-axis
K: +ve z-axis
O: -ve y-axis
P: +ve y-axis

Morgana controls (movement based on the initial direction he faces):
Up arrow: Forward
Down arrow: Backward
Left arrow: Left
Right arrow: Right
Space: Attack animation
G: Dance

Debug options:
Z: Disable axes
X: Enable axes
1: Enable face culling
2: Disable face culling
3: Fill mode
4: Wireframe mode
9: Disable light
0: Enable light

Other controls:
R: Reset character, camera, light position
