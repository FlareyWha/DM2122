How to play:
W: Move forward
A: Strafe left
S: Move backward
D: Strafe right
E: Interact when needed whenever text appears to do so
Mouse to move camera

Debug options:
F1: Enable debug mode (y wont be locked to 0 anymore so can fly around to check out map, also ignores boundaries)
F2: Disable debug mode
F3: Disable debug text
F4: Enable debug text
